## 页面定位

这一页聚焦网路／行动通讯相关的连接、处理、电源与接口器件，覆盖通信模组、网关、路由器、物联网节点与移动终端周边等方向，强调代表性应用而不是完整原厂列表。

## 代表性产品线与品牌

| 方向 | 代表性器件 | 代表性品牌示例 |
|------|------------|----------------|
| 无线连接 | Wi-Fi、BLE、Thread、Matter、Sub-1 GHz | 如 TI、Nordic、NXP |
| 有线网络 | Ethernet PHY、交换与接口周边 | 如 TI、Microchip、ADI |
| 电源管理 | 通信电源、PMIC、PoL 供电 | 如 MPS、TI、Richtek |
| 处理平台 | 网络 MCU、应用处理器、边缘 SoC | 如 NXP、ST、Renesas |
| 存储与缓存 | NOR Flash、DDR、eMMC | 如 Winbond、Micron、Samsung |

以上品牌仅作代表性示例，用于说明方案覆盖方向，不代表完整 line card。

## 典型应用场景

- 工业网关、蜂窝边缘终端与无线采集节点
- 路由器、接入点、智能中控与通信模组主板
- 蓝牙外设、可穿戴设备与低功耗 IoT 节点
- 需要本地缓存、协议桥接和多接口管理的联网设备

## 选型关注点

1. 无线协议、带宽、功耗和认证路径是否满足目标市场
2. 有线与无线混合组网下的接口数量和协议兼容性
3. 网络设备的电源效率、散热与长时间稳定运行能力
4. 终端是否需要本地 AI、缓存或安全启动能力
5. 模组化设计和后续替代料策略是否预留

## 与安路 FPGA / Xilinx 的协同方式

- 安路 FPGA 可做协议桥接、并行数据整理和自定义接口时序，外围使用 WT 文晔器件完成无线、PHY、电源与存储配套
- Xilinx 平台适合更高吞吐的网络加速、视频网关和边缘推理通信节点
- 在网关类产品中，用处理器承接协议栈和应用层，用 FPGA 处理低延迟数据通路与专用接口扩展
