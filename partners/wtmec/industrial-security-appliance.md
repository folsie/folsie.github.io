## 页面定位

这一页聚焦 WT 文晔在工控／安防／家电场景中的代表性方案方向。重点不是罗列完整代理清单，而是归纳常见的控制、电源、接口、传感与驱动器件组合，帮助系统设计更快落到可执行的 BOM。

## 代表性产品线与品牌

| 方向 | 代表性器件 | 代表性品牌示例 |
|------|------------|----------------|
| 控制与处理 | MCU、MPU、边缘控制 SoC | 如 ST、NXP、Renesas |
| 电源与驱动 | AC-DC、DC-DC、LDO、电机驱动 | 如 TI、MPS、Infineon |
| 连接与接口 | Ethernet PHY、USB、RS-485、CAN、隔离器 | 如 TI、Microchip、ADI |
| 感知与检测 | MEMS、温湿度、压力、成像周边 | 如 Bosch、ST、Sensirion |
| 存储与启动 | NOR Flash、DDR、eMMC | 如 Winbond、Micron、Samsung |

以上品牌仅作代表性示例，用于说明方案覆盖方向，不代表完整 line card。

## 典型应用场景

- PLC、HMI、工业网关与现场总线控制节点
- NVR、智能摄像机、门禁终端与边缘安防盒子
- 白电主控板、变频驱动板、触控面板与联网家电
- 工业采集终端、能源监测节点与楼宇自动化模块

## 选型关注点

1. 工业温度范围、长期供货周期与封装可替代性
2. 电源树设计是否满足多路时序、待机功耗与 EMI 要求
3. 接口协议是否覆盖 Ethernet、RS-485、CAN、USB 等系统边界
4. 安防与家电场景下的图像、音频、触控或电机控制需求是否需要专用器件协同
5. 量产阶段的 BOM 成本、交期与替代料策略是否提前纳入设计

## 与安路 FPGA / Xilinx 的协同方式

- 安路 FPGA 负责实时控制、接口桥接或并行采集，WT 文晔器件补齐电源、存储、PHY 与传感外围
- Xilinx 平台承担更高带宽视频、视觉或工业网络处理，WT 文晔方案用于配套 PMIC、DDR、Flash 与接口芯片
- 在安防和工业视觉场景中，用 FPGA 做预处理和时序控制，用外围 MCU / MPU 做系统管理与联网
