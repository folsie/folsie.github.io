## 如果你不知道先点哪一个

| 你的情况 | 优先建议 |
| --- | --- |
| 还没配基础保障 | 先看医疗险、意外险 |
| 家庭责任重 | 先看定期寿险 |
| 担心大病影响收入 | 先看重疾险 |
| 想做完整组合 | 先按医疗险 → 意外险 → 定寿 → 重疾险顺序看 |

## 我更建议的购买顺序

1. 先补医疗险，解决大额医疗费用问题
2. 再补意外险，成本低、基础性强
3. 有房贷和家庭责任的，尽早补定期寿险
4. 预算允许时，再补足重疾险责任

## 风险提示

- 保险产品最终以条款、健康告知、核保结论和投保页面展示信息为准。
- 不同产品的续保、免责、等待期和职业限制可能不同，投保前要单独确认。
- 如果你有既往病史、异常体检或高风险职业，建议先做核保评估再决定。
