## 进入链接前先看清楚

<div class="insurance-insight-panel">

## 如果不知道先看哪个

| 你的情况 | 优先看 |
| --- | --- |
| 还没有任何基础保障 | 医疗险、意外险 |
| 有房贷、孩子或赡养责任 | 定期寿险 |
| 担心大病后收入中断 | 重疾险 |
| 想做完整基础组合 | 医疗险 → 意外险 → 定期寿险 → 重疾险 |

## 风险提示

- 链接中的产品信息可能随保险公司和投保页面更新，投保前必须重新核对。
- 保险产品最终以条款、健康告知、核保结论和投保页面展示为准。
- 如有既往症、异常体检、住院史或高风险职业，建议先做核保评估。
