# 香港保险指标快照

更新时间：2026-06-08

## 说明

香港保险这部分不使用“平均收益率”做页面口径，改为使用香港保监局公开披露的分红实现率机制与市场统计数字，用来帮助理解产品类型和市场结构。

## 重点指标

| 日期 | 指标 | 数值 | 来源 |
| --- | --- | ---: | --- |
| 监管长期有效机制 | 分红实现率 | 按产品系列逐年披露 | 香港保险业监管局 |
| 2025-04-25 发布 | 2024 年长期业务新造保单保费 | 2,198 亿港元 | 香港保险业监管局临时统计数字 |
| 同上 | 非相连个人业务 | 2,081 亿港元 | 香港保险业监管局临时统计数字 |
| 同上 | 其中分红业务 | 1,824 亿港元 | 香港保险业监管局临时统计数字 |
| 同上 | 相连个人业务 | 112 亿港元 | 香港保险业监管局临时统计数字 |

## 页面使用建议

- 用“分红实现率”解释非保证利益的历史兑现情况，而不是宣传成未来保证回报。
- 讨论香港保险时，更应提示币种、跨境投保流程、长期持有要求与现金价值节奏。
- 如果后续要做更新，优先补年度或季度市场统计，以及分红实现率说明页的最新版本。
